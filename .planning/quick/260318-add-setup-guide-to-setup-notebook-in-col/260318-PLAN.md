Structure the lesson with these sections:

**Learning Objectives:**
- Understand what Google Colab is and why it is useful for learning Python
- Create a new Colab notebook from scratch
- Write and run Python code in Colab cells
- Understand the difference between code cells and text cells
- Save notebooks to Google Drive and share them

**Lesson Outline (4 parts):**

Part 1 — What is Google Colab? (~5 min)
- Free cloud-based Jupyter notebook service from Google
- No installation required — runs in the browser
- Pre-installed Python 3 + common libraries
- Saves to Google Drive automatically
- Use `<Tip>` callout: "You only need a Google account (Gmail) to get started. No credit card or software downloads needed."
- Use `<Info>` callout: "Colab is perfect for following along with lessons. You can run the exact same Python code you see in this course."

Part 2 — Creating Your First Notebook (~10 min)
- Step-by-step instructions:
  1. Go to colab.research.google.com
  2. Sign in with Google account
  3. Click "New notebook" (or File > New notebook)
  4. The notebook opens with an empty code cell
- Explain the Colab interface: toolbar, code cells, text cells, runtime indicator
- Use `<Warning>` callout: "Colab sessions disconnect after being idle. Your notebook is saved, but any variables or outputs are lost. Just click 'Run all' to restore your work."

Part 3 — Running Python Code (~10 min)
- How to type code in a cell and run it (Shift+Enter or click Play button)
- Show simple examples the student should type and run:
  ```python
  print("Hello from Google Colab!")
  ```
  ```python
  # Try some math
  2 + 2
  ```
  ```python
  name = input("What is your name? ")
  print(f"Welcome to Python, {name}!")
  ```
- Explain: output appears directly below the cell
- Adding new cells: "+ Code" button or Ctrl+M B
- Adding text cells for notes: "+ Text" button
- Use `<Tip>` callout: "Use text cells to write notes to yourself as you practice. It helps you remember what you learned!"

Part 4 — Saving and Organizing Your Work (~5 min)
- Notebooks auto-save to Google Drive under "Colab Notebooks" folder
- Rename notebook: click the title at top left
- Download as .py file: File > Download > Download .py
- Share with others: Share button (top right), works like Google Docs
- Use `<Info>` callout: "Create a folder called 'Python Course' in your Google Drive and move your Colab notebooks there to stay organized."

**Key Takeaways** section with 5-6 bullet points summarizing the lesson.

**Colab vs Local Python** comparison table:
| Feature | Google Colab | Local (VS Code) |
| Setup time | Instant | 30-60 minutes |
| Requires installation | No | Yes |
| Works offline | No | Yes |
| Saves automatically | Yes | Manual |
| Good for | Quick practice, sharing | Real projects, large programs |
