## Integration Issues

### P0 — Lesson pages 404 under unified slug

`generateStaticParams` in lesson page uses `getAllCourses()` emitting old section slugs. Never emits `courseSlug: 'python'`. All lesson URLs under `/courses/python/` return 404 with `dynamicParams = false`.

**Fix:** Replace `getAllCourses()` with `getUnifiedCourse().allLessons` in `generateStaticParams`, emitting `courseSlug: 'python'`.

### P0 — extractHeadings path uses wrong slug

`rawMdPath` uses `courseSlug` (= `'python'`) instead of `lesson.sourceCourseSlug` (= `'01-python-fundamentals'`). No `courses/python/` directory exists.

**Fix:** Replace `courseSlug` with `lesson.sourceCourseSlug` in `path.join`.

### P1 — Cross-section nav broken

`LessonNav` receives `course.lessons` (intra-section only). Last lesson in each section shows no Next button.

**Fix:** Thread `unifiedCourse.allLessons` into `LessonNav` instead of `course.lessons`.

### P2 — Stale /courses catalog page

Presents 12 separate courses with broken progress bars. No nav links to it.

**Fix:** Delete or redirect to `/courses/python`.

## Broken E2E Flows

1. **Lesson reading with ToC** — breaks at `extractHeadings` receiving empty content
2. **Cross-section navigation** — breaks at `LessonNav` showing no Next button at section boundaries
