<objective>
Animated concept mindmaps on every lesson page using ReactFlow, dagre auto-layout, and Motion entrance animations, with mindmap data auto-generated from lesson h2 headings.

Purpose: Give students a visual overview of each lesson's key concepts as an interactive, zoomable node graph with smooth entrance animations. Avoids manually authoring 120+ JSON files by auto-generating from existing lesson heading structure.

Output: ReactFlow mindmap component on every lesson page with per-lesson JSON data, build-time generation script, dagre layout, Motion staggered node entrance, and skeleton loading state.
</objective>

       - Import getAllCourses from lib/content
       - For each course, for each lesson, read the lesson MD file from `courses/{courseSlug}/{lessonSlug}.md`
       - Extract all h2 headings (## ...) from the MD content using regex `/^##\s+(.+)$/gm`
       - Generate a nodes array: root node (lesson title, id: "root") + one node per h2 heading (id: "node-{i}")
       - Generate edges array: root -> each h2 node
       - Write output to `courses/{courseSlug}/mindmaps/{lessonSlug}.json` with { nodes, edges } structure
       - Ensure mindmaps/ directory is created per course
       - Add `"generate:mindmaps": "tsx scripts/generate-mindmap-data.ts"` to package.json scripts
       - Add `tsx` as devDependency if not already present

       - Define `MindmapData` type: `{ nodes: Node[]; edges: Edge[] }` using @xyflow/react types
       - `getMindmapData(courseSlug: string, lessonSlug: string): MindmapData | null` — reads JSON from `courses/{courseSlug}/mindmaps/{lessonSlug}.json`, returns parsed data or null if file doesn't exist
       - `layoutMindmapData(nodes: Node[], edges: Edge[]): { nodes: Node[]; edges: Edge[] }` — uses @dagrejs/dagre to compute positions with rankdir:'TB', ranksep:60, nodesep:40, NODE_WIDTH:150, NODE_HEIGHT:40

    4. Run the generator script on the full course set to produce mindmap JSON files for all lessons

    5. Add `courses/*/mindmaps/*.json` to .gitignore (generated files, regenerated at build time)
