Pandas 2.x API notes for code correctness:
- DataFrame construction: `pd.DataFrame({'col': [...]})` or from dict/list of dicts
- Series construction: `pd.Series([...], name='col')`
- Label-based selection: `df.loc[row_label, col_label]` or `df.loc[condition, 'col']`
- Position-based selection: `df.iloc[0, 1]`
- Column access: `df['col']` (preferred) or `df.col` (avoid for ambiguity)
- Boolean filter: `df[df['col'] > 5]` or `df.query("col > 5")`
- dtypes: `df.dtypes`, `df['col'].astype(float)`
- Stats: `df.describe()`, `df['col'].mean()`, `df['col'].value_counts()`
- Shape/info: `df.shape`, `df.info()`, `df.head()`, `df.tail()`
- Rename: `df.rename(columns={'old': 'new'})`
- Sort: `df.sort_values('col', ascending=False)`

**lesson-04-indexing-and-selection.md** — "Indexing and Selection"
Duration: 50 minutes. Level: Intermediate.
Content:
- Part 1: Column selection — `df['col']` returns Series, `df[['col1', 'col2']]` returns DataFrame
- Part 2: `loc` — label-based row and column selection; `df.loc[0]`, `df.loc[0:3, 'name':'salary']`
- Part 3: `iloc` — position-based; `df.iloc[0]`, `df.iloc[0:3, 1:3]`
- Part 4: When to use loc vs iloc — loc for label access (after setting a custom index), iloc for positional slices

- Part 4: `.query()` method — string-based filtering; `df.query("salary > 50000 and department == 'Engineering'")`
