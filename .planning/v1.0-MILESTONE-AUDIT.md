### Integration Issue

**NotebookLMCard renders broken links:** `lib/notebook-urls.ts` contains PLACEHOLDER strings for 11 courses. The `NotebookLMCard` guard (`if (!url) return null`) checks for falsy values, but PLACEHOLDER strings are truthy — so the card renders a live "Open in NotebookLM" link that navigates to a non-existent notebook (404 at Google). **Affected requirement: NBLM-03.**
