# Section 1: Introduction to AI Code Agents

**Level:** Intermediate
**Duration:** 8 lessons x 30-40 minutes = ~5 hours total
**Prerequisites:** Python fundamentals, familiarity with software development concepts

---

## Section Overview

Before writing a single line of agent code, you need to understand the landscape: what AI code agents are, why they emerged, how they differ from simpler AI tools, and what mental models govern how they work.

This section is entirely conceptual. No code is required. The goal is to give you the vocabulary and mental framework that every subsequent section builds on. When you reach Section 2 and start using Claude Code, you will understand why each feature exists. When you reach Section 3 and write your first agent loop, you will know exactly what pattern you are implementing.

## What You Will Learn

By the end of this section, you will be able to:

- Define AI code agents and explain the agentic loop
- Categorize AI coding tools and describe when each category is appropriate
- Describe the Claude model family and its key capabilities
- Explain why large language models are capable of generating code despite predicting tokens
- Distinguish between copilot-style assistants and autonomous agents
- Describe the tool use paradigm: how agents connect LLM reasoning to real-world actions
- Explain the key safety considerations when deploying autonomous code agents
- Navigate the rest of this course with a clear picture of what you will build
