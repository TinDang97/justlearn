# AI Code Agents with Claude

**Level:** Intermediate
**Prerequisites:** Python fundamentals, basic API knowledge (HTTP requests, JSON)

## Course Description

Build AI-powered coding agents using Claude Code CLI, the Anthropic Messages and Agent API, and Model Context Protocol (MCP). Learn to automate code review, testing, documentation, and refactoring with AI agents that can read, write, and execute code autonomously.

This course takes you from understanding what AI code agents are, through hands-on practice with Claude Code CLI, up to building production-grade multi-agent pipelines. By the end, you will have shipped five real projects and understand the full agentic stack.

## Who This Course Is For

- Python developers who want to automate repetitive coding tasks with AI
- Engineers curious about how tools like GitHub Copilot and Cursor work under the hood
- Developers who want to build custom AI-powered developer tooling
- Anyone who wants to go beyond using AI chat and start building AI agents

## After This Course

You will be able to build AI agents that interact with codebases, execute tools, manage multi-turn conversations, and integrate with the broader MCP ecosystem — using Claude as the intelligence layer.
