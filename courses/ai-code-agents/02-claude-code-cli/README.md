# Section 2: Claude Code CLI

**Level:** Intermediate
**Duration:** 8 lessons x 35-45 minutes = ~5 hours total
**Prerequisites:** Section 1 complete; Node.js 18+; Anthropic API account

---

## Section Overview

This section moves from theory to practice. You will install Claude Code, configure it for your projects, and develop the workflow habits that make agentic coding consistently productive.

Claude Code is a CLI tool — it runs in your terminal, works with your existing editor, and integrates naturally into your current development workflow. There is no new IDE to learn. The skills you develop here apply immediately to real work.

## What You Will Learn

By the end of this section, you will be able to:

- Install and authenticate Claude Code on your development machine
- Run productive coding sessions with clear, effective prompts
- Manage context budget effectively across long tasks
- Use slash commands to optimize workflow efficiency
- Perform file operations, git operations, and complex multi-file tasks
- Write CLAUDE.md project instructions that encode your team's conventions
- Apply prompting patterns that consistently produce high-quality results
