# Course 01: Python Fundamentals - Getting Started

**Position in Curriculum:** Course 1 of 12 | **Level:** Absolute Beginner
**Duration:** 10 lessons x 2 hours = 20 hours total
**Prerequisites:** None - this is where everyone begins

---

## Course Description

Welcome to your very first Python course. If you've never written a single line of code before, you are in exactly the right place. This course treats you like an intelligent adult who simply hasn't learned programming yet - because that's exactly what you are.

Python is one of the world's most popular programming languages, used by everyone from NASA scientists to Instagram developers to data analysts at banks. It's known for being readable and beginner-friendly, which is why it's the perfect first language. The code looks almost like English.

In this course, you will go from "I don't know what programming is" to writing real programs that take input from a user, perform calculations, and display results. You'll set up your own development environment, understand how computers execute code, and build the foundation for everything that comes next.

By the end of this course, you'll have written your first 10+ programs and you'll be genuinely excited about what comes next.

---

## Learning Objectives

By the end of Course 01, students will be able to:

- Explain what programming is and what Python can be used for
- Install Python and VS Code and configure a working development environment
- Write, save, and run Python programs from the command line
- Use the `print()` function to display output
- Use the `input()` function to receive user input
- Perform basic arithmetic operations (+, -, *, /, //, %, **)
- Work with text (strings) using basic string operations
- Write readable code using comments and proper style
- Find and fix simple errors in Python programs
- Build a complete mini project using all course concepts

## What You'll Build

By the end of this course, students will have built:
- "Hello, World!" - the classic first program
- A greeting program that asks your name and says hello
- A simple tip calculator (enter bill amount, get tip suggestion)
- A unit converter (kilometers to miles, Celsius to Fahrenheit)
- A mini calculator with a menu interface
