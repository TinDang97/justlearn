# Course 2: Pandas Fundamentals

**Level:** Intermediate
**Duration:** 12 lessons x 30–60 minutes = ~7 hours total
**Prerequisites:** Section 1 (Introduction to Data Engineering) + Python fundamentals

---

## Course Overview

This section is where theory becomes code. You will write your first pandas programs, starting from the two fundamental data structures — Series and DataFrame — and building up to data selection, filtering, column transformations, type inspection, sorting, and statistics.

Every concept lesson includes interactive PracticeBlocks so you can run code directly in the browser. By the end of Section 2 you will be able to load, inspect, filter, and summarize tabular data using pandas — the foundation for everything in Sections 3–10.

## What You Will Learn

By the end of this section, you will be able to:

- Create and inspect pandas Series and DataFrames from Python data structures
- Select rows and columns using `loc`, `iloc`, and boolean filtering
- Filter DataFrames with compound conditions and the `.query()` method
- Add, modify, and drop columns; apply custom functions with `.apply()`
- Inspect and convert data types using `.dtypes`, `.info()`, and `.astype()`
- Sort data by single or multiple columns; get top-N rows with `.nlargest()`
- Compute summary statistics: `.describe()`, `.mean()`, `.value_counts()`, `.corr()`
- Inspect DataFrame structure with `.shape`, `.info()`, `.head()`, `.isnull().sum()`
- Rename columns and set a meaningful row index
