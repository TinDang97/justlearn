# Course 04: Functions & Modules

**Position in Curriculum:** Course 4 of 12 | **Level:** Beginner
**Duration:** 10 lessons x 2 hours = 20 hours total
**Prerequisites:** Courses 01-03

---

## Course Description

So far, you've been writing programs as one long sequence of instructions. Functions let you organize code into reusable, named blocks. They're the most fundamental tool for managing complexity in programming.

Think of functions as recipes. Once you write a recipe for chocolate cake, you don't write it again every time you want cake - you just reference the recipe. Functions work the same way: write the code once, use it everywhere.

This course teaches you to write functions that take inputs (parameters), produce outputs (return values), and maintain their own private scope. You'll also learn how Python's modules system lets you reuse code across projects - and how to tap into the vast standard library and third-party packages.

---

## Learning Objectives

By the end of Course 04, students will be able to:

- Define and call functions with `def`
- Use parameters to pass data into functions
- Use `return` to send data back from functions
- Apply default and keyword arguments
- Understand the LEGB scope rule fully
- Write lambda functions for simple operations
- Use Python's most useful built-in functions
- Create and import custom modules
- Install and use third-party packages with pip
