# Course 11: Automation & Scripting

**Level:** Advanced | **Duration:** 20 hours (10 lessons × 2 hours) | **Prerequisites:** Courses 1-10

---

## Course Overview

Learn to automate real-world tasks with Python: scheduling jobs, scraping web data, sending emails, interacting with spreadsheets, controlling the browser, and writing system administration scripts.

---

## What You'll Build

A **Daily Report Bot** that scrapes data, processes it, generates charts, and emails a formatted HTML report — all automatically on a schedule.
