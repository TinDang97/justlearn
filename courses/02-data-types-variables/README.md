# Course 02: Data Types & Variables

**Position in Curriculum:** Course 2 of 12 | **Level:** Beginner
**Duration:** 10 lessons x 2 hours = 20 hours total
**Prerequisites:** Course 01 - Python Fundamentals

---

## Course Description

In Course 1, you used variables and data almost without thinking about it. In Course 2, we go deeper. Understanding data types is like understanding ingredients in cooking - the same recipe (program) can produce very different results depending on what ingredients (data types) you use.

Python has a rich system of built-in data types, each with its own strengths and use cases. Numbers have different flavors (integers for counting, floats for measuring). Text is stored as strings with powerful manipulation capabilities. Booleans represent true/false logic that powers all decision-making. And Python has a special value called `None` that represents "no data."

This course also covers Python's type system - how to check what type a value is, convert between types safely, and understand why Python is called a "dynamically typed" language. You'll also learn about variable scope - a crucial concept that prevents bugs in larger programs.

---

## Learning Objectives

By the end of Course 02, students will be able to:

- Identify and use Python's four basic data types: int, float, str, bool
- Explain the difference between integers and floating-point numbers
- Apply string formatting techniques including f-strings and format methods
- Use boolean values and comparison operators correctly
- Convert between data types safely using int(), float(), str(), bool()
- Use `None` appropriately and check for it
- Apply naming conventions for variables and constants
- Understand basic variable scope (local vs global)
- Build data-driven programs with proper type handling
