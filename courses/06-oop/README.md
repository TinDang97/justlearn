# Course 06: Object-Oriented Programming

**Position in Curriculum:** Course 6 of 12 | **Level:** Intermediate
**Duration:** 10 lessons x 2 hours = 20 hours total
**Prerequisites:** Courses 01-05

---

## Course Description

Everything in Python is an object - strings, integers, lists, functions. But what does that mean? Object-Oriented Programming (OOP) is a way of organizing code around "objects" - entities that combine data and behavior into one self-contained package.

Think of it like this: in procedural programming, you have data (a student's name and grades) and functions (calculate average, display report) as separate things. In OOP, you create a `Student` object that knows its own name and grades, AND knows how to calculate its own average and display its own report.

This course teaches you to design and build classes - Python's mechanism for creating new types of objects. You'll learn encapsulation (hiding internal details), inheritance (creating specialized versions of classes), and polymorphism (writing code that works with many types).

OOP is the dominant paradigm in professional software development. After this course, you'll understand every framework and library you encounter.

---

## Learning Objectives

By the end of Course 06, students will be able to:

- Define classes with attributes and methods
- Use `__init__` to initialize objects
- Create and use instances of custom classes
- Apply encapsulation with properties and private attributes
- Implement inheritance to create class hierarchies
- Override methods in subclasses
- Use `super()` to call parent methods
- Understand polymorphism and duck typing
- Use Python's magic methods (dunder methods)
- Build a complete OOP application
