# Course 8: Working with Libraries

**Level:** Intermediate | **Duration:** 20 hours (10 lessons × 2 hours) | **Prerequisites:** Courses 1-7

---

## Course Overview

Learn to use Python's ecosystem of libraries. You'll master `pip`, virtual environments, and popular packages. After this course, you'll be comfortable installing any library, reading documentation, and integrating third-party packages into your projects.

---

## What You'll Build

A **Multi-tool CLI Application** that uses 5+ external libraries: fetches data from APIs, processes images, queries a SQLite database, sends emails, and handles command-line arguments.
