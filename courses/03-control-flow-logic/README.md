# Course 03: Control Flow & Logic

**Position in Curriculum:** Course 3 of 12 | **Level:** Beginner
**Duration:** 10 lessons x 2 hours = 20 hours total
**Prerequisites:** Courses 01-02

---

## Course Description

Every program you've written so far runs the same way every time - line by line, no variations. But real programs make decisions: "If the user is logged in, show the dashboard. Otherwise, show the login page." Real programs repeat actions: "Check each item in the shopping cart. Show the weather for the next 7 days."

Control flow is the heartbeat of programming. It's what makes programs intelligent and dynamic. In this course, you'll learn how to make your programs decide, choose, and repeat. By the end, you'll be able to write programs that respond meaningfully to any input.

This course covers Python's two fundamental control structures: **conditionals** (if/elif/else) for decision-making, and **loops** (while, for) for repetition. Together, these form the complete toolkit for controlling program execution.

---

## Learning Objectives

By the end of Course 03, students will be able to:

- Write if, if-else, and if-elif-else chains to make decisions
- Nest conditional statements appropriately
- Use logical operators (and, or, not) in conditions
- Write while loops with proper termination conditions
- Use for loops to iterate over sequences and ranges
- Control loops with break, continue, and pass
- Write nested loops for grid-like problems
- Recognize common loop patterns (accumulator, search, transformation)
- Build programs requiring both conditionals and loops
