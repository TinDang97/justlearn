# Course 10: Data Analysis & Visualization

**Level:** Intermediate-Advanced | **Duration:** 20 hours (10 lessons × 2 hours) | **Prerequisites:** Courses 1-9

---

## Course Overview

Learn to analyze real datasets using NumPy and Pandas, and visualize insights with Matplotlib and Seaborn. You'll work with CSV files, clean messy data, compute statistics, and create professional charts.

---

## What You'll Build

A **Sales Analytics Dashboard** — analyze a real sales dataset to find trends, top products, regional performance, and visualize it all in publication-quality charts.

---
