# Course 7: File Handling & Exceptions

**Level:** Intermediate | **Duration:** 20 hours (10 lessons × 2 hours) | **Prerequisites:** Courses 1-6

---

## Course Overview

Learn to read and write files, handle errors gracefully with try/except, work with common file formats (CSV, JSON), and use context managers. These skills are essential for any real-world Python program.

---

## What You'll Build

By the end of this course, you'll build a **Personal Journal Application** that stores journal entries in JSON, reads them back, searches through entries, and handles all errors gracefully.

## Key Libraries

- `pathlib` — modern file path handling
- `csv` — CSV file reading/writing
- `json` — JSON serialization
- `os` — operating system interface
- `contextlib` — context manager utilities
