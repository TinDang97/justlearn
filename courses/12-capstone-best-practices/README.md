# Course 12: Capstone Projects & Best Practices

**Level:** Advanced | **Duration:** 20 hours (10 lessons × 2 hours) | **Prerequisites:** Courses 1-11

---

## Course Overview

The final course synthesizes everything learned. You'll master professional development practices (testing, code quality, documentation, architecture), then choose and build one of three complete capstone projects. This is where you become a working Python developer.

---

## What You'll Build

Choose ONE of three capstone projects:
1. **Blog Platform** — full-stack Flask web app with auth, REST API, admin panel
2. **Data Analytics Pipeline** — automated ETL + dashboard + scheduled reporting
3. **CLI Developer Tool** — professional command-line tool with plugins and config
