# Course 9: Web Development Basics

**Level:** Intermediate-Advanced | **Duration:** 20 hours (10 lessons × 2 hours) | **Prerequisites:** Courses 1-8

---

## Course Overview

Build web applications with Python using Flask. You'll understand how the web works (HTTP, requests/responses), create HTML templates, handle forms, build a REST API, and connect everything to a database.

---

## What You'll Build

A **Task Management Web App** — a full CRUD web application with user accounts, task lists, REST API, and a simple frontend.

---
